# Rightm_Flask_API


## Getting Started

Python 3.6.5

### Installing and Running
git clone https://github.com/Japolk/rightm_flask.git

create and activate virtualenv (optional)

install required modules from requirements.txt

in console:
```
export FLASK_APP=api
flask init-db
flask run 
```
